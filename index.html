Stocky: MapReplenisher

Política de Privacidad

Última actualización: 26 de abril de 2026

Stocky respeta tu privacidad. Esta aplicación almacena todos los datos localmente en tu dispositivo y no recopila información personal en servidores externos.

1. Información que recopilamos

Stocky es una aplicación de uso offline. Los datos que introduces (nombres de tiendas, reponedores, rutas, fotos y ubicaciones) se almacenan exclusivamente en tu dispositivo utilizando una base de datos local (SQLite).

No recopilamos, almacenamos ni transmitimos información personal a servidores externos.

2. Permisos de la aplicación

Stocky solicita los siguientes permisos para su funcionamiento:

Permiso Propósito ¿Datos enviados externamente?
Ubicación (GPS) Mostrar tu ubicación en el mapa y registrar la ubicación de tiendas No
Cámara Capturar fotos de tiendas con overlay de información No
Internet Cargar mapas (OpenStreetMap), geocoding (Nominatim) y mostrar anuncios Solo solicitudes de mapas y anuncios
Almacenamiento Guardar y exportar archivos (PDFs, archivos .stocky) No

3. Servicios de terceros

3.1 Google AdMob

Stocky utiliza Google AdMob para mostrar anuncios publicitarios. AdMob puede recopilar ciertos datos del dispositivo para servir anuncios relevantes, incluyendo:

  • Identificador de publicidad del dispositivo
  • Dirección IP
  • Información de actividad en la aplicación

Puedes consultar la política de privacidad de Google AdMob en: Política de Privacidad de AdMob

Para optar por no recibir anuncios personalizados, puedes desactivar el identificador de publicidad en la configuración de tu dispositivo Android.

3.2 OpenStreetMap y Nominatim

Utilizamos los servicios de OpenStreetMap para mostrar mapas interactivos y Nominatim para búsquedas de direcciones. Estas solicitudes se envían a los servidores de OpenStreetMap conforme a su política de privacidad.

4. Almacenamiento de datos

Todos tus datos (tiendas, reponedores, rutas, fotos, campos personalizados) se almacenan:

  • Localmente en la base de datos SQLite de tu dispositivo
  • En el sistema de archivos de tu dispositivo (fotos y PDFs generados)

No hacemos copias de seguridad de tus datos en servidores externos. Eres responsable de utilizar la función de exportación (.stocky) para respaldar tu información.

5. Compartir datos

Tus datos solo se comparten cuando tú decides hacerlo activamente:

  • Al usar la función Exportar para generar un archivo .stocky
  • Al usar la función Compartir PDF para enviar reportes
  • Al usar la función Compartir fotos del sistema

Ninguno de estos datos pasa por nuestros servidores.

6. Datos de menores

Stocky no está dirigida a menores de 13 años. No recopilamos conscientemente datos de menores.

7. Seguridad

Dado que todos los datos se almacenan localmente en tu dispositivo, la seguridad de tu información depende de la seguridad de tu dispositivo. Recomendamos:

  • Mantener tu dispositivo con un bloqueo de pantalla activo
  • No compartir tu dispositivo con personas no autorizadas
  • Realizar copias de seguridad periódicas usando la función de exportación

8. Tus derechos

Dado que todos los datos están en tu dispositivo, tienes control total sobre ellos:

  • Acceso: Puedes ver todos tus datos dentro de la aplicación
  • Eliminación: Puedes eliminar datos individuales o desinstalar la aplicación para eliminar todo
  • Exportación: Puedes exportar tus datos usando la función de exportación (.stocky)

9. Cambios a esta política

Podemos actualizar esta política de privacidad ocasionalmente. Cualquier cambio será reflejado en la fecha de "Última actualización" y en el contenido de esta página.
